# 🌤️ Weather App - Real-time Weather Fetcher

A Python application that fetches real-time weather data for any city using the **OpenWeatherMap API**. Displays temperature, humidity, wind speed, and weather conditions.

---

## 📌 Features

- 🌡️ Real-time temperature (°C)
- 💧 Humidity percentage
- 🌬️ Wind speed
- ☁️ Weather condition description
- 🌍 Works for any city worldwide
- ⚠️ Handles errors (invalid city, no internet, wrong API key)

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python 3.x | Core language |
| requests | HTTP API calls |
| OpenWeatherMap API | Weather data source |
